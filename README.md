# Bill Sheba - ISP Management Platform

Bill Sheba is a comprehensive management platform designed specifically for **Internet Service Providers (ISPs)**. It streamlines the complex task of managing subscribers, handling billing, and monitoring connectivity in one centralized professional dashboard.

## 🚀 Key Features

- **Subscriber Management**: Efficiently track internet users, manage their profiles, and handle account statuses.
- **Billing & Payments**: Automate billing cycles, track paid and unpaid bills, and manage payment records (including partial payments).
- **Package Management**: Create and assign various internet speed tiers and plans to different users.
- **Connectivity Monitoring**: Real-time insights into active user sessions and session history.
- **Professional Dashboard**: Clean, modern interface for managing the day-to-day operations of an ISP business.

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- [pnpm](https://pnpm.io/) or [bun](https://bun.sh/)

### Installation

First, run the development server:

```bash
bun dev
# or
pnpm dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 💻 Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Stan-js
- **Database/Query**: TanStack Query & DB
- **UI Components**: Radix UI & Lucide Icons
